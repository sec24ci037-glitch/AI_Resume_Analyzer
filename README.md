# AI Resume Analyzer

## About the Project

AI Resume Analyzer is a web application developed using Python and Flask. The main purpose of this project is to help users analyze their resumes by extracting text from uploaded resume files using OCR technology. Based on the extracted content, the application provides basic feedback about the resume.

This project was developed as a learning project to understand Flask, OCR, and web application development.

---

## Features

- Upload resume files
- Extract text using Tesseract OCR
- Display extracted text
- Basic resume quality analysis
- Simple and easy-to-use interface

## How to Run

1. Clone the repository.

```
git clone https://github.com/sec24ci037-glitch/AI_Resume_Analyzer.git
```

2. Move into the project folder.

```
cd AI_Resume_Analyzer
```

3. Create a virtual environment.

```
python -m venv venv
```

4. Activate the virtual environment.

```
venv\Scripts\activate
```

5. Install the required packages.

```
pip install -r requirements.txt
```

6. Start the application.

```
python app.py
```

7. Open your browser and visit:

```
http://127.0.0.1:5000
```
